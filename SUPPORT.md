# Support

This project is provided as-is for educational and personal use.

**Support for this controller and for installation assistance will not be provided.**

If you have experience with TouchOSC and djay Pro and would like to contribute improvements, feel free to open a pull request.

For general questions about TouchOSC, refer to the [Hexler documentation](https://hexler.net/touchosc/manual).

For general questions about djay Pro MIDI mapping, refer to the [Algoriddim support documentation](https://help.algoriddim.com/user-manual/djay-pro-mac/midi/mapping).
